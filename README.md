Java + Selenide + JUnit 5

Цель теста - проверить, что пользователь может пройти основной путь оценки и в конце увидеть сообщение об успешной отправке заявки: Ваша заявка успешно отправлена!

Сценарий покрывает следующие шаги:

открывает карточку автомобиля по фиксированному URL

нажимает кнопку "Оцените ваш автомобиль"

в модальном окне выбирает "Оцените онлайн"

заполняет номер автомобиля

проходит шаги формы (выбор пробега, переход к оценке, выбор "Получить оценку")

заполняет контактные данные

подтверждает согласие

отправляет заявку

проверяет успешный результат по тексту в финальном окне

Тест реализован в стиле Page Object Model, чтобы код был поддерживаемым:

CarDetailPage - действия на карточке автомобиля

EvaluationStartPage - шаги формы оценки

SubmissionSuccessPage - проверка результата

SubmitCarEvaluationTest - сценарий
